## Building an Operational Intelligence Platform for Global Storage Infrastructure

### Situation
Storage reporting, monitoring, and administration were distributed across multiple systems and manual processes.

### Challenge
Create a centralized platform to collect data, generate reports, and support storage administration.

### Actions
- Developed scheduled Python services to gather infrastructure data.
- Built SQL databases for storage metrics and reporting.
- Created a web interface using PHP, JavaScript, HTML, and Microsoft IIS.
- Integrated Python automation for administration and maintenance tasks.
- Aggregated information from NetApp systems, SAN switches, Windows hosts, and Linux servers.

### Results
- Enabled leadership to make decisions using live operational data instead of reports that were 3–4 days old.
- Centralized reporting and operational visibility.
- Improved efficiency of storage administration workflows.
- Supported operations across 50+ systems in 15+ global locations.
- Reduced reliance on manual reporting and data collection.

### Technologies
Python, SQL, PHP, JavaScript, HTML, Microsoft IIS, NetApp, Brocade, PowerShell, Bash
